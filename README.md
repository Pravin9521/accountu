# Accountu – Khatabook-like Accounting App

This repository contains a full-stack accounting application similar to Khatabook, designed to be production-ready, secure, and deployable on free tiers (Render/Railway for backend, Vercel/Netlify for frontend, and MongoDB Atlas).

## Project Structure

- `backend/` – Node.js + Express API (JWT auth, organizations, ledger, interest, reminders, dashboard).
- `frontend/` – React + Vite application (Tailwind-ready UI, dashboards, ledger views).

## Getting Started

### Backend

```bash
cd backend
npm install        # already run once, repeat if needed
npm test           # placeholder from npm init
```

### Frontend

```bash
cd frontend
npm install        # already run once, repeat if needed
npm run build      # basic build to verify setup
```

Feature-specific setup (JWT, MongoDB, Tailwind configuration, dashboards, etc.) will be implemented in later tasks and commits.
